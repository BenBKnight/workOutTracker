## Fitness Tracker
  ****
  ### Description
  Fitness tracking application that utilizes MongoDB to track and record user workouts.
  
  ### Installation
  Utilize npm install to install node packages
  ```
 npm i 
```
  ### Usage
  Rund node server.js for local usage, or deployed appilication.
  ```
 node server.js 
```
